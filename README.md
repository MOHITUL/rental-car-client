DriveNow

Live Site:https://drive-now-project-96.netlify.app/

Welcome to DriveNow, a modern and user-friendly platform designed to simplify car rentals and bookings. Our website provides seamless experiences for users to browse, book, and manage vehicles efficiently.

Features

Browse and Search Cars: Easily explore available cars with filters for type, model, and rent price.  

Book Cars Instantly: Users can book their desired car in just a few clicks and get real-time booking confirmation.  

User Dashboard: Manage your bookings, view booking history, and check car availability from a personalized dashboard.  

Add and Manage Listings: Car owners can add new cars, update details, and manage their listings effortlessly.  

Responsive Design: Fully responsive layout for mobile, tablet, and desktop devices ensuring a smooth user experience.  

Secure Authentication: Sign up and log in safely using email/password or social login options.  

Tech Stack

- React.js for the frontend  
- Tailwind CSS for styling  
- Fetch API for client-server communication  
- Firebase for authentication
- Swiperslide for slider
- Lucide-react & react-icon for icons
- Toastify & sweetalert for alert
